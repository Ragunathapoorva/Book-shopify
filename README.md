Book-Shopify

Book-Shopify is a static online bookstore website developed using HTML, CSS, and JavaScript.  
The project demonstrates core frontend web development concepts with a focus on clean layout, usability, and basic interactivity.

 Overview

This project simulates a simple book shopping interface where users can browse books in a structured and visually organized manner.



 Key Features

- Structured and readable UI  
- Responsive page layout  
- Basic client-side interactivity  
- Simple and maintainable codebase  



 Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)
